## 카카오 로그인 API
- 카카오 REST API 키 발급 및 작동원리 참고 : https://velog.io/@jiwoow00/Spring-boot-JWT-이용한-백엔드-카카오-로그인
- 실행 순서
![alt text](<카카오로그인동작순서.jpg>)

1. 로그인 화면에서 카카오로그인 클릭
2. (프론트) LoginView.vue 에서 함수 실행 리다이렉트uri를 설정해서 카카오에 로그인 요청 보냄
3. (프론트) 카카오에서 요청 처리 후 인가코드와 함께 KakaLoginView.vue로 리다이렉트
4. (프론트) 리다이렉트된 곳에서 인가코드를 백엔드 서버로 전송
5. (백) 백엔드 서버에서는 프론트에서 받은 인가코드를 카카오 서버에 보내면서 엑세스토큰 요청
6. (백) 카카오서버에서 받은 엑세스토큰으로 사용자 정보 조회 요청 → 응답은 JSON타입이고, 사용자 정 User 객체에 담기
7. (백) DB에서 전체 유저 가져와서 등록된 이메일인경우 (이전에 카카오로그인했던 이메일) → 로그인처리해서 프론트에 응답 전송
8. (백) 등록 안된 이메일인 경우 DB user 테이블에 insert 하고 로그인처리해서 프론트에 응답 전송
9. (프론트) 프론트는 백 응답받고 로그인 처리 팝업띄움
